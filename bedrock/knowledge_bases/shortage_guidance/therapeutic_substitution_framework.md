# Therapeutic Substitution Framework

_Source: ASHP Policy Positions on Therapeutic Interchange, Joint Commission Medication Management Standards, APhA Principles of Practice for Pharmaceutical Care_

---

## Overview

Therapeutic substitution (also called therapeutic interchange) is the dispensing of a chemically different drug expected to produce the same therapeutic outcome as the originally prescribed medication. During drug shortages, pharmacists must evaluate alternatives within structured frameworks.

**Critical Disclaimer:** This document provides GENERAL FRAMEWORKS ONLY for pharmacist professional review. It does NOT provide specific drug-to-drug substitutions. All therapeutic decisions require individual clinical assessment by a qualified healthcare professional.

**FOR PHARMACIST REVIEW ONLY**

---

## Principles of Therapeutic Substitution

### Core Requirements

Any therapeutic substitution during a shortage must satisfy ALL of the following:

1. **Same Therapeutic Class** — The substitute belongs to the same pharmacological class
2. **Same Indication** — The substitute is FDA-approved (or has strong evidence) for the same indication
3. **Equivalent Efficacy** — Published evidence supports comparable clinical outcomes
4. **Acceptable Safety Profile** — No unacceptable new risks for the patient population
5. **P&T Committee Authorization** — Pre-approved interchange protocol OR emergency authorization documented

### Levels of Substitution

| Level | Type                    | Description                                    | Authorization Required                    |
| ----- | ----------------------- | ---------------------------------------------- | ----------------------------------------- |
| 1     | Generic Substitution    | Same active ingredient, different manufacturer | Standard pharmacy practice                |
| 2     | Therapeutic Interchange | Different drug, same class, same indication    | P&T protocol or prescriber approval       |
| 3     | Therapeutic Alternative | Different class, same therapeutic goal         | Prescriber order required                 |
| 4     | Off-Label Alternative   | Drug used outside FDA-approved indication      | Prescriber order + clinical justification |

### Decision Framework

```
Step 1: Identify the clinical need
  → What condition is being treated? What is the therapeutic goal?

Step 2: Review same-class options
  → Are other agents in the same class available?
  → Is there head-to-head efficacy data?

Step 3: Assess patient-specific factors
  → Allergies, drug interactions, renal/hepatic adjustments, route constraints?

Step 4: Verify authorization pathway
  → Does an approved P&T interchange protocol exist?
  → If not, obtain prescriber authorization and document rationale

Step 5: Implement with monitoring
  → Appropriate dose conversion, monitoring parameters, follow-up scheduled?
```

## When to Involve Clinical Pharmacists vs. Physicians

### Clinical Pharmacist Authority (with P&T protocol)

Pharmacists may independently manage interchanges when:

- An approved P&T interchange protocol exists for the specific substitution
- Patient has no contraindications per protocol exclusion criteria
- Clinical scenario matches protocol scope (indication, population, setting)
- State regulations permit pharmacist-initiated therapeutic interchange

### Physician Involvement Required

Escalate to prescribing physician when:

- No P&T interchange protocol exists
- Patient has exclusion criteria (complex comorbidities, contraindications)
- Proposed alternative is a different therapeutic class (Level 3)
- High-risk medications involved (narrow therapeutic index)
- Off-label use is being considered (Level 4)
- Previous substitution attempt had suboptimal outcome

### Urgency-Based Collaboration

```
LOW (days available):  Pharmacist reviews → Presents recommendation →
                       Physician approves → Implement
MODERATE (24-48 hrs):  Pharmacist selects per protocol → Implements →
                       Notifies physician within 24 hrs
HIGH (immediate):      Pharmacist implements best available →
                       Immediate physician notification → Reassess together
```

## Special Populations Considerations

### Pediatric Patients

- Require pediatric clinical pharmacist or pediatrician review for all interchanges
- Verify age-appropriate formulation is available (liquid vs. tablet, palatability)
- Confirm dosing reference specific to pediatric population
- Consider excipient safety (benzyl alcohol, propylene glycol contraindicated in neonates)
- Enhanced monitoring frequency compared to adult protocols

### Geriatric Patients

- Screen alternative against AGS Beers Criteria and STOPP/START criteria
- Perform comprehensive drug interaction check including OTC and supplements
- Prefer once-daily dosing when therapeutically equivalent options exist
- Assess fall risk with certain medication classes
- Consider caregiver ability to manage new regimen

### Pregnant and Lactating Patients

- Require OB/GYN or maternal-fetal medicine consultation for all interchanges
- Verify alternative against current pregnancy safety databases
- Document risk-benefit discussion with patient
- For lactating patients, verify infant exposure levels are acceptable

## Documentation Requirements

### Minimum Documentation for Every Therapeutic Interchange

Each substitution must include in the patient medical record:

1. **Original medication** — Drug name, dose, route, frequency, indication
2. **Reason for change** — Shortage status, unavailability documentation
3. **Alternative selected** — Drug name, dose, route, frequency
4. **Clinical rationale** — Why this alternative was selected
5. **Authorization** — P&T protocol reference OR prescriber approval date/time
6. **Monitoring plan** — Parameters, frequency, expected outcomes
7. **Patient notification** — Date patient was informed
8. **Pharmacist identification** — Name and credentials

### Institutional Records

Maintain aggregate records for quality assurance:

- Total interchanges by therapeutic category per reporting period
- Outcomes tracking (efficacy maintained, adverse events, treatment failures)
- Prescriber notification compliance rates
- Time from shortage detection to protocol activation

## Key Principles Summary

| Principle            | Guidance                                                     |
| -------------------- | ------------------------------------------------------------ |
| Patient Safety First | Never compromise safety for convenience or cost              |
| Evidence-Based       | Base decisions on published clinical evidence                |
| Authorized           | Operate within institutional protocols and scope of practice |
| Documented           | Record every decision with complete rationale                |
| Monitored            | Follow up to verify therapeutic outcomes                     |
| Communicated         | Inform all stakeholders: prescriber, patient, care team      |
| Reversible           | Plan for transition back when shortage resolves              |
